# React with TypeScript

To run the React example, run the following:

```
cd react-app
npm install
npm run start
```

Then, open a browser to http://localhost:3000


# Svelte

To run the Svelte example, run the following:

```
cd svelte
npm install
npm run dev
```

Then, open a browser to http://localhost:8080
